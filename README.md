# Apollo-Station-
2D sci-fi metroidvania set aboard a quarantined space station.
# Story
A maintenance engineer gets trapped when the station enters quarantine mode. The station's AI activates a sterilization protocol, turning everything on board hostile to all living things. Marked as a threat, the protagonist must survive deadly scanning lights and find a way home.
#Core Mechanic
Hazard-based traversal — navigating through environmental dangers, with deadly scanning laser beams as the signature obstacle.
# Gameplay Loop
Explore → Avoid hazards → Defeat enemies → Reach a new sector → Escape the station.
# Tech Stack
Engine: Unreal Engine 5.
Art: Adobe Photoshop + DragonBones.
Audio: Epidemic Sound.
No AI tools were used — all content was created and modified by the team.
# Context
Built in 30 hours during a GameDev Hackathon.
